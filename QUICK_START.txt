┌───────────────────────────────────────────────────────────────┐
│  STEP 1: SETUP (One-time only)                               │
└───────────────────────────────────────────────────────────────┘

   Double-click this file:
   📄 setup-backend.bat

   This will:
   ✓ Install all dependencies
   ✓ Create configuration files
   ✓ Set up folder structure


┌───────────────────────────────────────────────────────────────┐
│  STEP 2: GET API KEY                                         │
└───────────────────────────────────────────────────────────────┘

   1. Open browser and go to:
      🔗 https://makersuite.google.com/app/apikey

   2. Click "Create API Key"

   3. Copy the key (looks like: AIzaSyD...)

   4. Open this file with Notepad:
      📄 backend\.env

   5. Paste your key:
      GEMINI_API_KEY=AIzaSyD_your_actual_key_here


┌───────────────────────────────────────────────────────────────┐
│  STEP 3: START SERVER                                        │
└───────────────────────────────────────────────────────────────┘

   Double-click this file:
   📄 start-backend.bat

   You should see:
   ✓ 🚀 Server running on http://localhost:5000
   ✓ 🔑 Gemini API Key: ✅ Configured


┌───────────────────────────────────────────────────────────────┐
│  STEP 4: TEST                                                │
└───────────────────────────────────────────────────────────────┘

   Open browser and visit:
   🔗 http://localhost:5000/api/health

   Should show: "status": "ok"


┌───────────────────────────────────────────────────────────────┐
│  STEP 5: USE IT!                                             │
└───────────────────────────────────────────────────────────────┘

   1. Keep backend server running
   2. Start your React app (npm run dev)
   3. Go to PPT Converter page
   4. Upload a PPT file
   5. Watch the magic! ✨


╔═══════════════════════════════════════════════════════════════╗
║                      DAILY USAGE                              ║
╚═══════════════════════════════════════════════════════════════╝

   Every time you want to use the PPT converter:

   1. Double-click: start-backend.bat
   2. Start your React app
   3. Upload PPTs and enjoy!


╔═══════════════════════════════════════════════════════════════╗
║                     TROUBLESHOOTING                           ║
╚═══════════════════════════════════════════════════════════════╝

   ❌ "npm is not recognized"
   → Install Node.js from https://nodejs.org/

   ❌ "Port 5000 already in use"
   → Change PORT=5001 in backend\.env

   ❌ "GEMINI_API_KEY not set"
   → Check backend\.env file has your API key

   ❌ CORS error
   → Restart both frontend and backend
